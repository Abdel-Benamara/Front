Modélisation SVIR

Le modèle SVIR est un modèle épidémiologique qui consiste à diviser la population en trois catégories :
- S : les personnes Susceptibles d'être inféctées
- V : les personnes Vaccinées
- I : les personnes Infectées
- R : les personnes Retirées (guéries ou décédées)
Ce modèle repose sur la résolution d'équations différentielles basée sur S, V, I, R ainsi que certains indicateurs tels que le taux de reproduction du virus r0, le taux de guérison, le taux de transmission du virus ou encore le taux de vaccination afin de déterminer l'évolution de la propagation du virus dans le temps.
Il nous permet de prédire le nombre cumulé de personnes vaccinées par la COVID-19
