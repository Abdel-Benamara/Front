Modélisation Machine Learning

Le modèle de Machine Learning utilisé ici est un modèle de régression linéaire basé sur les 13 variables suivantes :

  • x1 : le nombre de décès quotidiens
  • x2 : le nombre de guérisons quotidiennes
  • x3 : le nombre de décès quotidiens en Ehpad
  • x4 : le total de personnes en réanimation
  • x5 : le nombre de nouvelles infections confirmées
  • x6 : le total de personnes hospitalisées
  • x7 : le nombre de dépistage de la COVID-19 réalisés quotidiennement
  • x8 : le nombre de dépistage de la COVID-19 positifs quotidiennement
  • x9 : le nombre de nouvelles infections confirmées en Ehpad
  • x10 : le nombre d'entrées quotidiennes en réanimation
  • x11 : le nombre de nouvelles hospitalisations quotidiennes
  • x12 : le nombre de nouvelles premières injections de vaccin contre la COVID-19
  • x13 : le dernier taux de reproduction du virus en date (aussi appelé r0)

Ce modèle nous permet de prédire le nombre de personnes qui seront quotidiennement testées positives à la COVID-19 dans les jours à venir.

Modélisation Machine Learning

Le modèle de Machine Learning utilisé ici est un modèle de régression linéaire basé sur les 11 variables suivantes :

  • x1 : le stock du nombre total de doses de vaccin
  • x2 : le stock du nombre de doses de vaccin Pfizer-BioNTech
  • x3 : le stock du nombre de doses de vaccin Moderna
  • x4 : le stock du nombre de doses de vaccin Pfizer-BioNTech réservé aux Ehpad
  • x5 : le nombre de nouvelles premières injections de vaccin
  • x6 : le total de prises de rendez-vous hebdomadaire pour vaccination
  • x7 : le nombre de prises de rendez-vous hebdomadaire pour injection de rang 1
  • x8 : le nombre de prises de rendez-vous hebdomadaire pour injection de rang 2
  • x9 : le nombre total de livraisons hebdomadaires de doses de vaccin
  • x10 : le nombre de livraisons hebdomadaires de doses de vaccin Pfizer-BioNTech
  • x11 : le nombre de livraisons hebdomadaires de doses de vaccin Moderna

Ce modèle nous permet de prédire le nombre de premières doses qui seront injectées quotidiennement dans les jours à venir.
